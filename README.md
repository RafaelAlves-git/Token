# Projeto Token – Delphi
Aplicação desktop desenvolvida em **Delphi (VCL)** com o objetivo de gerar e gerenciar **senhas dinâmicas (tokens)** baseadas em **data e hora do sistema**.  
O sistema foi projetado para uso interno, oferecendo praticidade, rapidez e segurança na geração de diferentes tipos de senhas operacionais.

##  Funcionalidades
-  Geração automática de senhas com base na data e hora atual
-  Diferentes tipos de senhas
-  Cópia rápida da senha para a área de transferência com um clique
-  Atualização manual dos tokens via botão de refresh
-  Feedback visual interativo (hover e destaque de painéis)
   Lógicas de cálculo independentes para cada tipo de senha

## Tecnologias Utilizadas
- **Delphi (VCL)**
- Componentes nativos (`TPanel`, `TLabel`, `TImage`)
- Manipulação de data e hora (`DecodeDate`, `DecodeTime`)
- Área de transferência do Windows (`Clipbrd`)
- Programação orientada a eventos
